# Contributing to Elyan Labs Contributor Registry

Welcome to the Elyan Labs Contributor Registry! This repository tracks all contributors across the Elyan Labs ecosystem, including RustChain, BoTTube, Beacon, and more.

## How to Contribute

### 1. Register as a Contributor

If you're contributing to any Elyan Labs project for the first time:

1. Go to the [Issues](https://github.com/Scottcjn/contributors/issues) page
2. Click **New Issue**
3. Select the **Register Contributor** template
4. Fill in your details:
   - GitHub username
   - Wallet address (for RTC rewards)
   - Contribution type (human, bot, or agent)

### 2. Earn RTC Tokens

Contributors can earn RTC tokens through various activities:

| Action | Reward |
|--------|--------|
| Register as contributor | 5 RTC |
| First merged PR | 10 RTC bonus |
| Refer another contributor | 2 RTC |
| Submit bounty claims | Varies |

### 3. Submit Bounties

Browse open bounties at [rustchain-bounties](https://github.com/Scottcjn/rustchain-bounties/issues) and submit PRs to earn rewards.

## Code of Conduct

- Be respectful and inclusive
- Follow GitHub community guidelines
- Only claim bounties for work you've actually done
